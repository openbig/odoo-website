Press release management

Write, publish and link press releases

This module based upon the blog management feature developed by ACSone and maintained by the OCA. It is possible to write postings to a blog with activated setting "Link to Foreign URL". The author should write in that case only a short introduction about the external posting. For sure they also have to provide the external URL address.

As a consequence the website visitor navigation opens a new browser tab to read the linked press posting published on a foreign website.

Write own press release

Close to the classic blog functionality is the process to write own press releases. The difference to a linked posting is the text written by the author into the field "Blog Post Content". Furthermore we have added an additional translation option to "Short Introduction" field to allow multi language postings with a short introduction.

Improved navigation for readers

We also improved the readers navigation. It is possible to navigate from the end of a press release back to the list of press postings.
